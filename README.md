# Como instalar o projeto

## Pré-requisitos

- [Node.js](https://nodejs.org/en/)

## Como instalar

- Clone o repositório

```bash
$ git clone
```

- Entre na pasta do projeto

```bash
$ cd
```

- Instale as dependências

```bash
$ npm install
```

- Rode o projeto

```bash
$ npm start
```

- Acesse o projeto no seu navegador em:

```
http://localhost:4200
```
